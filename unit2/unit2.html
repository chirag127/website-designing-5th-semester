Elements of HTML

HTML is made up of elements. An element is a part of a web page that is defined by a tag. Tags are used to tell the browser how to display the content. The most common tags are the opening tag and the closing tag. The opening tag is the name of the element surrounded by angle brackets. The closing tag is the same as the opening tag, but with a forward slash before the name of the element. The content of the element is placed between the opening and closing tags. The content is the information that is contained within the element. The content can be text, images, videos, or other HTML elements. The following is an example of an HTML element.

Opening tag: < h1>

Closing tag: < /h1>

HTML Attributes

HTML attributes are used to provide additional information about an element. Attributes are placed inside the opening tag. The following is an example of an HTML attribute.

Opening tag: < a href="https://www.w3schools.com/html/default.asp">

Closing tag: < /a>

Content: HTML Tutorial

href = "https://www.w3schools.com/html/default.asp" is an attribute. The attribute is used to specify the link address.

There are two types of attributes. The first type is a required attribute. Required attributes must be included in the opening tag. The second type is an optional attribute. Optional attributes do not have to be included in the opening tag. The following is a list of the most common HTML attributes.

href = "https://www.w3schools.com/html/default.asp" (required)

src = "https://www.w3schools.com" (required)

Working with Lists

lists are the most common way to display information in a structured way.

There are two types of lists: ordered and unordered.

Ordered lists

Ordered lists are lists where each item in the list is numbered.

Attributes of Ordered lists

The ordered list has the following attributes: 

    reversed - Specifies that the list order should be descending (9,8,7...).

    start - Specifies the start value of an ordered list.

    type - Specifies the type of marker to use in the list. The default value is 1. Other possible values are: a, A, i, I.

Working with Tables

The HTML < table> tag defines an HTML table.

Each table row is defined with a < tr> tag. Each table header is defined with a < th> tag. Each table data/cell is defined with a < td> tag.

By default, the text in < th> elements are bold and centered.

By default, the text in < td> elements are regular and left-aligned.

Table attributes

The table element has the following attributes: 

    align - Specifies the horizontal alignment of a table. Possible values are: left, center, right.

    bgcolor - Specifies the background color of a table.

    border - Specifies the width of the border around a table.

    cellpadding - Specifies the space between the cell wall and its content in a table.

    cellspacing - Specifies the space between cells in a table.

    frame - Specifies which parts of the frame surrounding the table should be visible. Possible values are: void, above, below, hsides, lhs, rhs, vsides, box, border.

    rules - Specifies which rules to draw between cells. Possible values are: none, groups, rows, cols, all.

    summary - Specifies a summary of the content of a table.

    width - Specifies the width of a table.

Example of Table attributes

The following example shows a table with the align attribute set to center, the bgcolor attribute set to #FF0000, the border attribute set to 6, the cellpadding attribute set to 10, the cellspacing attribute set to 20, the frame attribute set to box, the rules attribute set to groups, the summary attribute set to "This is an example of a table element", and the width attribute set to 600.

Tables and Frames

The HTML < frame> tag defines a window (a frame) in a frameset.

The HTML < frameset> tag defines a set of frames.

The HTML < noframes> tag defines an alternate content for users that do not support frames.

Frame attributes

The frame element has the following attributes: 

    longdesc - Specifies a URL to a document that contains a long description of the frame.

    name - Specifies a name for the frame.

    noresize - Specifies that the frame should not be resizable.

    scrolling - Specifies whether or not to display scrollbars in a frame. Possible values are: yes, no, auto.

    src - Specifies the URL of the document to display in a frame.

Example of Frame attributes

The following example shows a frame with the longdesc attribute set to "http://www.w3schools.com/html/frame_longdesc.asp", the name attribute set to "mainframe", the noresize attribute set to true, the scrolling attribute set to "no", and the src attribute set to "http://www.w3schools.com".

Frameset attributes

The frameset element has the following attributes: 

    cols - Specifies the number of columns in a frameset.

    rows - Specifies the number of rows in a frameset.

Example of Frameset attributes

Working with Hyperlinks

The HTML < a> tag defines a hyperlink, which is used to link from one page to another.

The HTML < area> tag defines an area inside an image-map (an image-map is an image with clickable areas).

The HTML < link> tag defines the relationship between a document and an external resource (most used to link to style sheets).

A attributes

The a element has the following attributes: 

    charset - Specifies the character encoding for a linked document.

    coords - Specifies the coordinates of a link.

    download - Specifies that the target will be downloaded when a user clicks on the hyperlink.

    href - Specifies the URL of the page the link goes to.

    hreflang - Specifies the language of the linked document.

    media - Specifies on what device the linked document will be displayed.

    name - Specifies a name for the target.

    rel - Specifies the relationship between the current document and the linked document.

    rev - Specifies the relationship between the linked document and the current document.

    shape - Specifies the shape of a link.

    target - Specifies where to open the linked document.

    type - Specifies the media type of the linked document.

Example of A attributes

The following example shows an a with the charset attribute set to "UTF-8", the coords attribute set to "0,0,100,100", the download attribute set to "true", the href attribute set to "http://www.w3schools.com", the hreflang attribute set to "en", the media attribute set to "screen", the name attribute set to "test", the rel attribute set to "alternate", the rev attribute set to "stylesheet", the shape attribute set to "rect", the target attribute set to "_blank", and the type attribute set to "text/html".

Images and Multimedia

The HTML < img> tag defines an image in an HTML page.

The HTML < map> tag is used with < area> elements to define an image-map (an image-map is an image with clickable areas).

The HTML < audio> tag defines sound, such as music or other audio streams.

The HTML < video> tag defines a video, such as a movie clip or other video streams.

Img attributes

The img element has the following attributes: 

    alt - Specifies an alternate text for an image, if the image cannot be displayed.

    crossorigin - Allows for the image to be fetched from another origin without being tainted by
    the cross-origin.

    height - Specifies the height of an image.

    ismap - Specifies an image as a server-side image-map.

    longdesc - Specifies a URL to a detailed description of an image.

    src - Specifies the URL of an image.

    usemap - Specifies an image as a client-side image-map.

    width - Specifies the width of an image.
