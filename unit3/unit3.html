Concept of CSS

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.

Creating Style Sheet

There are three ways of inserting a style sheet:

  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

An external style sheet can be written in any text editor. The style sheet file should not contain any HTML tags. The style sheet file must be saved with a .css extension. To use an external style sheet, add a link to the external style sheet file in the head section of an HTML document.

example of external css

<head> <link rel="stylesheet" type="text/css" href="mystyle.css"> </head>

Internal CSS

An internal style sheet may be used if one single HTML page has a unique style. The internal style is defined inside the style element, inside the head section.

example of internal css

<head> <style> body {background-color: powderblue;} h1 {color: blue;} p {color: red;} </style> </head>

Inline CSS

An inline style may be used to apply a unique style for a single element. To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

example of inline css

<h1 style="color:blue;text-align:center;">This is a heading</h1> <p style="color:red;">This is a paragraph.</p>

CSS syntax

A CSS rule-set consists of a selector and a declaration block:

selector {
property: value;
}

CSS selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

There are many different types of CSS selectors, which will be introduced in the following chapters.

Types of CSS selectors

There are many different types of CSS selectors:

  1. Simple selectors (select elements based on name, id, class)
    1. element selector

      element selector selects all elements with the specified name.

      example

      p {
      color: red;
      }

    2. id selector

      id selector selects the element with a specific id.

      example

      #para1 {
      color: red;
      }

    3. class selector

      class selector selects all elements with a specific class.

      example

      .center {
      text-align: center;
      color: red;
      }

    4. universal selector

      universal selector selects all elements.

      example

      * {
      color: red;
      }

    5. grouping selector

      grouping selector selects all elements with the same style definitions.

      example

      h1, h2, p {
      color: red;
      }

    6. descendant selector

      descendant selector selects elements that are descendants of a specified element.

      example

      p a {
      color: red;
      }

    7. child selector

      child selector selects elements that are the children of a specified element.

      example

      ul > li {
      color: red;
      }

  2. Attribute selectors (select elements based on an attribute or attribute value)
    1. attribute selector

      attribute selector selects elements with a specified attribute.

      example

      a[title] {
      color: red;
      }

    2. attribute value selector

      attribute value selector selects elements with a specified attribute value.

      example

      a[title="w3schools"] {
      color: red;
      }

    3. attribute begins with selector

      attribute begins with selector selects elements with a specified attribute value, beginning with a specific substring.

      example

      a[href^="https"] {
      color: red;
      }

    4. attribute ends with selector

      attribute ends with selector selects elements with a specified attribute value, ending with a specific substring.

      example

      a[href$=".pdf"] {
      color: red;
      }

    5. attribute contains selector

      attribute contains selector selects elements with a specified attribute value containing a specific substring.

      example

      a[href*="w3schools"] {
      color: red;
      }

    6. first child selector

      first child selector selects the first element among a group of siblings.

      example

      p:first-child {
      color: red;
      }

    7. link selector

      link selector selects all unvisited links.

      example

      a:link {
      color: red;
      }

    8. visited selector

      visited selector selects all visited links.

      example

      a:visited {
      color: red;
      }

    9. active selector

      active selector selects the active link.

      example

      a:active {
      color: red;
      }

    10. hover selector

      hover selector selects links on mouse over.

      example

      a:hover {
      color: red;
      }

    11. focus selector

      focus selector selects the input element which has focus.

      example

      input:focus {
      color: red;
      }

    12. target selector

      target selector selects the current active #id.

      example

      #news:target {
      color: red;
      }

    13. enabled selector

      enabled selector selects input elements that are enabled.

      example

      input:enabled {
      color: red;
      }

    14. disabled selector

      disabled selector selects input elements that are disabled.

      example

      input:disabled {
      color: red;
      }

    15. checked selector

      checked selector selects input elements that are checked.

      example

      input:checked {
      color: red;
      }

    16. not selector

      not selector selects all elements that are not matched by the specified selector.

      example

      :not(p) {
      color: red;
      }

    17. empty selector

      empty selector selects all elements that have no children (including text nodes).

      example

      p:empty {
      color: red;
      }

    18. lang selector

      lang selector selects elements based on the language of the element.

      example

      p:lang(fr) {
      color: red;
      }

    19. root selector

      root selector selects the document's root element.

      example

      :root {
      color: red;
      }

Css properties

css properties are used to set the style of an element.

example

p {
color: red;
}

css properties types

css properties are of two types

  1. shorthand properties
  2. longhand properties

shorthand properties

shorthand properties are used to set the values of multiple properties at once.

example

p {
margin: 0px 0px 0px 0px;
}

longhand properties

longhand properties are used to set the values of a single property.

example

p {
margin-top: 0px;
margin-right: 0px;
margin-bottom: 0px;
margin-left: 0px;
}

css properties categories

css properties are of two categories

  1. inherited properties
  2. non inherited properties

inherited properties

inherited properties are properties that are inherited by the child elements.

example

p {
color: red;
}

non inherited properties

non inherited properties are properties that are not inherited by the child elements.

example

p {
background-color: red;
}

Box Model

The box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

margin

The margin property is used to create space around elements, outside of any defined borders.

margin: 0px;

margin: 10px;

margin: 10px 20px;

margin: 10px 20px 30px;

margin: 10px 20px 30px 40px;

border

The border property is used to specify the width, style, and color of an element's border.

border: 0px;

border: 10px;

border: 10px 20px;

border: 10px 20px 30px;

border: 10px 20px 30px 40px;

padding

The padding property is used to generate space around an element's content, inside of any defined borders.

padding: 0px;

padding: 10px;

padding: 10px 20px;

padding: 10px 20px 30px;

padding: 10px 20px 30px 40px;
